SQLiteChangeSetConflictResult 枚举

This enumerated type represents the result of a user-defined conflict resolution callback.

命名空间: System.Data.SQLite
程序集: Native.Tool.dll
语法
public enum SQLiteChangeSetConflictResult

字段

名称 描述
Abort

If this value is returned, any changes applied so far are rolled back and the call to Apply(SessionConflictCallback, SessionTableFilterCallback, Object) will raise a SQLiteException with an error code of Abort.
Omit

If a conflict callback returns this value no special action is taken. The change that caused the conflict is not applied. The application of changes continues with the next change.
Replace

This value may only be returned from a conflict callback if the type of conflict was Data or Conflict. If this is not the case, any changes applied so far are rolled back and the call to Apply(SessionConflictCallback, SessionTableFilterCallback, Object) will raise a SQLiteException with an error code of Misuse.

If this value is returned for a Data conflict, then the conflicting row is either updated or deleted, depending on the type of change.

If this value is returned for a Conflict conflict, then the conflicting row is removed from the database and a second attempt to apply the change is made. If this second attempt fails, the original row is restored to the database before continuing.
