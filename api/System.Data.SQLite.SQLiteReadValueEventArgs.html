SQLiteReadValueEventArgs 类

This class represents the parameters and return values for the GetBlob(Int32, Boolean), GetBoolean(Int32), GetByte(Int32), GetBytes(Int32, Int64, Byte[], Int32, Int32), GetChar(Int32), GetChars(Int32, Int64, Char[], Int32, Int32), GetDateTime(Int32), GetDecimal(Int32), GetDouble(Int32), GetFloat(Int32), GetGuid(Int32), GetInt16(Int32), GetInt32(Int32), GetInt64(Int32), GetString(Int32), and GetValue(Int32) methods.

ExtraEventArgs

If the GetBytes(Int32, Int64, Byte[], Int32, Int32) or GetChars(Int32, Int64, Char[], Int32, Int32) method is being called, this object will contain the array related parameters for that method. If the GetBlob(Int32, Boolean) method is being called, this object will contain the blob related parameters for that method.
