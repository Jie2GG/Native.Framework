SQLiteReadValueCallback 委托

This represents a method that will be called in response to a request to read a value from a data reader. If an exception is thrown, it will cause the data reader operation to fail -AND- it will continue to unwind the call stack.

命名空间: System.Data.SQLite
程序集: Native.Tool.dll
语法
public delegate void SQLiteReadValueCallback(SQLiteConvert convert, SQLiteDataReader dataReader, SQLiteConnectionFlags flags, SQLiteReadEventArgs eventArgs, string typeName, int index, object userData, out bool complete);
参数
类型 名称 描述
SQLiteConvert convert

The SQLiteConvert instance in use.
SQLiteDataReader dataReader

The SQLiteDataReader instance in use.
SQLiteConnectionFlags flags

The flags associated with the SQLiteConnection instance in use.
SQLiteReadEventArgs eventArgs

The parameter and return type data for the column being read from the data reader.
System.String typeName

The database type name associated with this callback.
System.Int32 index

The zero based index of the column being read from the data reader.
System.Object userData

The data originally used when registering this callback.
System.Boolean complete

Non-zero if the default handling for the data reader call should be skipped. If this is set to non-zero and the necessary return value is unavailable or unsuitable, an exception will be thrown.
