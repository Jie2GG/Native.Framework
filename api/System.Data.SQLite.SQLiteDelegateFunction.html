SQLiteDelegateFunction 类

This class implements a SQLite function using a System.Delegate. All the virtual methods of the SQLiteFunction class are implemented using calls to the SQLiteInvokeDelegate, SQLiteStepDelegate, SQLiteFinalDelegate, and SQLiteCompareDelegate strongly typed delegate types or via the System.Delegate.DynamicInvoke(System.Object[]) method. The arguments are presented in the same order they appear in the associated SQLiteFunction methods with one exception: the first argument is the name of the virtual method being implemented.
