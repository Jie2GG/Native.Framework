SQLiteValue 类

This class represents a value from the SQLite core library that can be passed to the sqlite3_value_*() and associated functions.

Persist()

Uses the native value handle to obtain and store the managed value for this object instance, thus saving it for later use. The type of the managed value is determined by the type affinity of the native value. If the type affinity is not recognized by this method, no work is done and false is returned.

声明
public bool Persist()
返回
类型 描述
System.Boolean

Non-zero if the native value was persisted successfully.
